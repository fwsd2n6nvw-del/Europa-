# Moon Grid Alien

Simple mobile-friendly browser game starter.

## Files

- `index.html` — page layout only
- `style.css` — visual design, grid, buttons, animations
- `map.js` — map constants, terrain, crater tiles, glow tiles, blocked feature areas
- `sprites.js` — SVG sprite functions
- `game.js` — movement, controls, board startup, action pulse, feature placement

## Current Additions

- Crashed UFO at x=9, y=8
- UFO footprint is 3 tiles wide by 4 tiles tall
- UFO is decorative but also blocks movement on those tiles
- Action pulse is larger and red with a retro sensor vibe

## Mobile / GitHub Pages Notes

This project uses plain HTML, CSS, and JavaScript.

No npm.
No build tools.
No external libraries.

Upload all files together to GitHub Pages.
