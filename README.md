# Moon Grid Alien

Simple mobile-friendly browser game starter.

## Files

- `index.html` — page layout only
- `style.css` — visual design, grid, buttons, animations
- `map.js` — map constants, terrain, crater tiles, glow tiles
- `sprites.js` — SVG sprite functions
- `game.js` — movement, controls, board startup, action pulse

## Mobile / GitHub Pages Notes

This project uses plain HTML, CSS, and JavaScript.

No npm.
No build tools.
No external libraries.

Upload all files together to GitHub Pages.
