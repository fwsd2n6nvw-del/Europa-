# Moon Grid 40x40 Starter

A basic GitHub Pages compatible 40x40 grid game foundation set on a moon landscape.

## Files

- `index.html` — the full playable/displayable page in one file.

## How to use on GitHub from iPhone

1. Unzip the project.
2. Upload `index.html` to a GitHub repository.
3. In GitHub, open repository Settings.
4. Go to Pages.
5. Set the branch to `main` and folder to `/root`.
6. Save and open the generated GitHub Pages link.

## Current elements

- 40x40 grid.
- Lunar terrain colors.
- Craters.
- Ridge rings around craters.
- Tap/click tile selection.
- Selected coordinate display.
